## 📂 Highlighted Projects

Here are a few representative projects that showcase my skills:

* **Walmart Holiday Sales Analysis**
  Created an ETL pipeline (Python & Pandas), layered medallion architecture (bronze / silver / gold), and analyzed holiday vs non-holiday sales trends.

* **Voltmart Orders Cleaning (PySpark)**
  Built parallel cleaning pipelines (PySpark DataFrame API + SQL), validated consistency across both, and produced clean Parquet datasets for ML downstream.

* **E-Commerce Sales Forecasting (PySpark ML)**
  Aggregated daily sales, engineered time features, trained a Random Forest model in Spark, evaluated using MAE, and forecasted demand for a target week.
